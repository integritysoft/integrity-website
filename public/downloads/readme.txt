==================================
Integrity Assistant 0.1.2 Public Alpha
==================================

Thank you for downloading Integrity Assistant, the AI-powered digital companion that understands your screen activity and helps you navigate your digital world.

INSTALLATION INSTRUCTIONS
------------------------

Windows:
1. Extract all files from the ZIP archive
2. Run the install.bat file
3. Follow on-screen instructions to complete the installation
4. A desktop shortcut will be created automatically

macOS:
1. Extract all files from the ZIP archive
2. Open Terminal and navigate to the extracted folder
3. Run the following command: bash install.sh
4. Follow on-screen instructions to complete the installation

Linux:
1. Extract all files from the ZIP archive
2. Open Terminal and navigate to the extracted folder
3. Run the following command: bash install.sh
4. Follow on-screen instructions to complete the installation
5. A desktop launcher will be created automatically (on most desktop environments)

SYSTEM REQUIREMENTS
------------------
- Windows 10/11, macOS 10.15+, or Ubuntu 20.04+ (other Linux distros may work)
- Python 3.8 or newer
- 4GB RAM minimum (8GB recommended)
- 500MB disk space
- Internet connection for initial setup and AI processing

ACCOUNT INFORMATION
------------------
- You will need to create an account when first launching Integrity Assistant
- Each account is limited to 100 questions per day
- All data is processed securely and your privacy is our top priority
